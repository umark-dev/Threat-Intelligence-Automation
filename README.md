# Threat Intelligence Automation (IP Reputation Checker)

## Description
This project automates threat intelligence lookups by checking IP addresses against AbuseIPDB to determine whether they are malicious or safe.

## Features
- API-based IP reputation checking
- Identifies malicious IPs using threat intelligence data
- Generates automated reports
- Simulates SOC threat enrichment workflow

## Technologies
- Python
- REST API
- Threat Intelligence
- Cybersecurity Automation

## Setup
1. Get free API key from AbuseIPDB
2. Replace API key in script

## How to Run
python threat_checker.py

## Output
Results saved in report.txt
